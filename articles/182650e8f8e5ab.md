## 第4問

難易度:★★☆☆☆
```java:Main.java

    public static void main(String[] args) {
        int[] score = {91, 10, 80 ,7 ,4};
        for (int i = 0; i < score.length; i++) {
```

問題2:`System.out.println(score[i]);`についている`[]`は何と言うか？

:::details 回答

:::

## 第7問
言語:Java
難易度:★★★★★
```java:Main.java
public class Main {
    public static void main(String[] args) {
        System.out.println("乱数作成をします。");
        System.out.printkn("乱数は、0〜100までです。");
        System.out.printkn("キーボードで入力してください。");
        int r = new java.util.Random().nextInt(101);
        int answer = new java.util.Scanner(System.in).nextInt();
        int diff = 0;
        diff = answer - r;
        diff = Math.ads(diff);
        if (r == answer) {
            System.out.println("ピッタリ！！");
            System.out.println("差は\"" + diff + "\"です！！");
        } else {
            System.out.println("残念。。。差は" + diff + "\"です！！");
        }
    }
}
```
問題:`System.out.println("乱数を作成します。");`についている`.`は何と言うか？

:::details 回答
ドット
:::

問題2:`diff = answer - r;`についている`-`は何と言うか？

:::details 回答
ハイフン
:::

問題3:`System.out.println("差は\"" + diff + "\"です！！");`についている`\`は何と言うか？

:::details 回答
バックスラッシュ
:::
