# デザインの意味
まず、 **「デザインとはなんなのか？」** ということについて説明します。

:::details デザインとは？
> 「デザイン」は日本では「意匠」と訳されていました。意匠の意味を調べてみると、
> > 絵画・詩文や催し物などで、工夫をめぐらすこと。趣向。
美術・工芸・工業製品などで、その形・色・模様・配置などについて加える装飾上の工夫。
>
> と出てきます。
この意味の通り「物の見た目の綺麗さやオシャレさ」のことを「デザイン」と言い、デザイナーとは「物の見た目を綺麗にしたりオシャレにしたりする仕事」と言うことができます。これに関しては多くの方が納得できるのではないでしょうか。


https://ait-design.com/column/whats-design-vol-1-デザインの意味

:::

つまりは、**色や形、模様や配置などの装飾の工夫**のことを指しています。

# UI/UXについて
UI/UXについての説明が不要な方は、[デザインの理想図](https://zenn.dev/individual_blog/articles/815a9d5cce08f2#デザインの理想図)からご覧ください。
## UIとは？
続いて、UIについて説明します。

:::details UIとは？
> UIとは「User Interface（ユーザー インターフェイス）」の略称です。Interfaceには、本来「接点、接触面」という意味があります。

https://jp.ext.hp.com/techdevice/technologysc/creator_002/

:::

さらに、UIには**CUI**と**GUI**の2つの種類があります。

:::details CUIとは？
> CUIとは、Character User Interface（キャラクター ユーザー インターフェイス）の略となります。これはキーボード入力を用いたインターフェイスのことであり、当然、コンピューターが生まれた当時のUIはこのCUIのみでした。コンピューターが生まれた当時は、現代で主流となっているグラフィカルなデザインを持った直感的に操作できるようなUIはまだ生まれておらず、画面に表示されるのは文字列のみだったのです。つまり、UIはいわゆるコマンドライン方式に相当するもののみでした。そのため、操作は非常に難しく、現代のように誰でも使えるものではなく、コンピューターを使いこなせるのは専門知識を持った専門家のみだったのです。

https://jp.ext.hp.com/techdevice/technologysc/creator_002/

:::

:::details GUIとは？
> GUIとは、Graphical User Interface（グラフィカル ユーザー インターフェイス）の略となります。画面に表示されるのは文字列のみ、入力デバイスはキーボードのみというCUIには、グラフィカルな要素がなく使い方が直感的にわからないという弱点がありました。その弱点を克服するための画期的な発明が、カーソル操作です。これによって、UIに初めて視覚的なインターフェイスが誕生しました。現代では当たり前の操作となっている「必要なフォルダをカーソルで選択する」「ドラッグして複数のファイルを同じフォルダに入れる」といった操作が可能となったのです。
>
> GUIの登場によって、コンピューターの使いやすさは格段に向上し、専門家でなくても簡単にパソコンを使えるようになりました。また、コンピューターマウスやサーチボックス、ハイパーテキストなどの、現代では当然のように存在するインターフェイスも、GUI、ひいてはカーソル操作がなければ生まれなかったインターフェイスだと言えるでしょう。

https://jp.ext.hp.com/techdevice/technologysc/creator_002/

:::

## UXとは？
UXについて説明します。

:::details UXとは？
> UXとは、「User eXperience（ユーザー エクスペリエンス）」の略で、ユーザーがサービスや製品の使用で得られる体験を示す言葉です。例えばECサイトでは「欲しい商品を見つけやすい」「サイトの機能を理解しやすい」「サイトのデザインがオシャレで好ましい」「カスタマーサポートの対応が早くて親切」などと感じる、サービス・製品にまつわる体験の全てがUXにあたります。

https://www.nec-solutioninnovators.co.jp/sp/contents/column/20230505_uxui.html

:::

```-:
⚠️よくある間違い⚠️

❌(間違い1) UIとUXは同じ
❌(間違い2) UXでwebやアプリの価値が決まる（あるいはUIでwebやアプリの価値が決まる）
```
