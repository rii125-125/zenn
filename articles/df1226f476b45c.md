# はじめに
どうも！！rii_125です。
タイトルにもあるように新しく **[広報ブログ](https://zenn.dev/p/individual_blog)** を立ち上げました！！
（広報ブログは、publicationというZennの機能を使って実装しています。）
この記事では、立ち上げた経緯と今後の目的について解説していきます。

# 広報ブログとは？
広報ブログとは何か？簡単にご説明します。

> 広報ブログは、自分の記事を気軽に投稿できるようにしたpublicationです。
> 自分に関する記事を広報ブログでは簡単に投稿して、自分の記事を投稿するときのモチベーションをあげよう！！と開設者の思いがあり、広報ブログが開設されました。

と、いうように広報ブログは個人個人の記事を投稿できるような記事となっています。

広報ブログは現在チーム募集中です。ぜひ、ご参加ください！！


# 立ち上げた経緯
きっかけは、記事の人気度です。
トレンドの記事ってすごいですよね。やっぱり参考になるものばかり。
でも、伸びない人は（自分も含めて）どうやったらみんなに見てもらえるか？どうしたらみんなにいい記事として認めてもらえるか？そんなに簡単に伸びない。だったら、少しでも気軽に記事を投稿できて、仲間と協力して記事を気軽に作れる**広報ブログ**を作りました。

広報ブログでは、参加者が気軽に記事を投稿して一緒に盛り上げていこう！！と考えています。
もし、参加してくれる方がいればぜひチームへの参加お願いします。

# 今後の目的
広報ブログは、今後合同でブログを書いて楽しくZennの活動を続けられるようにしていきます。
そのために、以下のことを目標として活動をしていきます。

- 誰でも安心して投稿できる場所にしたい。
- 合同で記事を作成して、達成感を得られるような場所にしたい。
- 近い将来Zennによる活動をより多く活発化させたい。

これらのことを目標として、これからも頑張っていきます。

# 最後に

と、いうことで今後とも広報ブログをよろしくお願いします。🙇‍♂️
